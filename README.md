# 🏥 Healthcare Analytics Dashboard

A professional **data analytics dashboard project** focused on analyzing healthcare data to uncover insights related to **patient demographics, billing trends, and hospital resource utilization**.

---

## 📌 Project Overview

This project aims to simulate a **real-world healthcare data analysis workflow** by performing:

- Data cleaning and validation  
- Exploratory data analysis (EDA)  
- Visualization and dashboard creation  
- Insight generation for decision-making  

The dashboard helps identify patterns in **patient age groups, disease distribution, billing behavior, and hospital stay durations**.

---

## 🎯 Objectives

- Analyze patient demographics across age groups  
- Identify high-frequency medical conditions  
- Understand billing patterns and detect anomalies  
- Evaluate hospital stay duration across diseases  
- Generate actionable insights for healthcare management  

---

## 🛠️ Tech Stack

- **Python (Pandas, NumPy, Matplotlib)**  
- **Jupyter Notebook**  
- **Power BI**  
- **CSV Dataset**  
- **GitHub**  

---

## 📊 Dashboard Features

- 📌 KPI Cards (Total Patients, Avg Billing, Avg Stay)
- 👩‍⚕️ Age Group Distribution Analysis  
- 🩺 Disease Frequency Analysis  
- 💰 Billing Trends & Comparisons  
- 🚨 Outlier Detection (Box Plot)  
- 🛏 Hospital Stay Analysis  
- 📈 Clean and interactive Power BI dashboard

## 📈 Key Insights

- The **60+ age group** contributes the highest number of patients, indicating increased healthcare demand among senior citizens.  
- **Working-age groups (21–60)** show consistent patient distribution across categories.  
- Certain medical conditions exhibit **higher billing trends**, indicating cost-intensive treatments.  
- Outlier analysis highlights **high-value cases**, which may represent critical or specialized treatments.  
- Some diseases require **longer hospital stays**, reflecting higher resource utilization.  

---

## 💡 Business Recommendations

- Increase focus on **geriatric healthcare services**  
- Optimize hospital resources for **high-demand age groups**  
- Investigate **high billing outliers** for cost control  
- Improve planning for **long-duration treatments**  
- Use data-driven strategies for **efficient healthcare management**
